# Explore The World (REST Countrie API)

## Overview
Explore The World is an interactive web application that allows users to explore countries around the world using dynamic map interface. The application intergrate REST countries API and Leaflet.js to display country data. Users can search for countries, filter by region, and track visited countries using persistant local storage.

## Features
-Interactive world map using Leaflet.js
-Search by country name functionality
-Region filtering (Africa, Americas, Asia, Europe, Oceania)
-Country information panel including region, capital, population, and flags
-Visted feature with localStorage

## Technologies Used
-HTML
-CSS
-JavaScript
-REST Countries API
-Leaflet.js 
-LocalStorage
